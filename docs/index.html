- About -

学科や学年の垣根を超え「こんなものが欲しい」「こんなものを作りたい」をみんなで協力して 実現していくサークルです。
チーム制作やコンテスト活動でのメンバー募集、課題制作や自己制作の進捗報告の場としても活用出来ます。

【活動場所】オンライン、機械・電気電子研究室 6階(C2-W602)杉村研究室など。
【活動内容】サークル内でプロジェクトを立ち上げ、自由に参加,見学,意見交換などを行います。
